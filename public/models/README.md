# Rack 3D model

- **Default:** `rack.glb` (auto-generated by `npm run generate:model` — includes procedural brushed-metal / panel / grille / floor textures and detailed geometry)
- **Component node names** must match `id` in `public/data/components.json`:
  `switch-1`, `router-1`, `server-1`, `gpu-server-1`, `storage-1`, `power-1`, `cabling-1`, `cooling-1`
- **Frame** node: `rack-frame` (not clickable)

Replace `rack.glb` with your own Blender/exported asset to upgrade visuals; keep the same node names for click detection.
